WP Post Activity

## Description
A lightweight plugin designed to track administrative actions and post engagement. 
It provides real-time monitoring of content changes and detailed view statistics directly in dashboard.


This plugin serves two main purposes:
1. **Activity Logging:** Plugin logs creating, updating, or deleting post activity.
2. **View Tracking:** Plugin counts how many times each post is viewed by visitors.


## Installation
1. Upload the `wp-post-activity` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **'Plugins'** menu in WordPress.


## How to Use
* **View Logs:** Navigate to the **"Post Activity -> Activity Logs"** menu in dashboard to see a history of who added, edited or deleted specific posts.
* **Check Stats:** Visit the **"Post Activity -> Post Views"** page to see a table of your most popular posts.
* **Front-end Display:** Add shortcode `[post_views_count]` inside your posts to display a view count on the post page.


## Requirements
**PHP** - 8.0+
**WordPress** - 6.0+
**MySQL** - 8.0+
